Array in Generale (Fissi e C-Style)
Esercizi di Base
Reverse: Crea un array fisso di 10 interi. Scrivi un ciclo che inverta l'ordine degli elementi (il primo diventa l'ultimo, ecc.) senza usare un array di supporto.

Ricerca: Dato un array di 20 numeri casuali, scrivi un programma che trovi il valore massimo e la sua posizione (indice).

Frequenza: Crea un array di 10 elementi (valori da 1 a 5). Conta quante volte compare il numero 3.


Passaggio a Funzione: Scrivi una funzione stampaArray(int arr[], int size) che stampi tutti gli elementi. Spiega perché il parametro size è obbligatorio.

Esercizi "Trabochetto"
Indice Fuori Limite: Cosa succede se dichiari int a[5] e provi a scrivere a[5] = 100? Scrivi il codice e osserva se il compilatore ti ferma o se il programma crasha a runtime.

L'inizializzazione parziale: Se scrivi int a[100] = {1, 2, 3};, cosa contengono le celle dalla posizione 3 alla 99? Dimostralo con un ciclo di stampa.

Copia Impossibile: Prova a copiare un array in un altro usando l'operatore = (es: arr2 = arr1). Spiega l'errore del compilatore e scrivi la soluzione corretta usando un ciclo.


Decadimento Costante: Crea un array const int a[5]. Prova a passare a a una funzione che accetta int*. Il compilatore accetta? Perché la const-correctness è fondamentale qui?
