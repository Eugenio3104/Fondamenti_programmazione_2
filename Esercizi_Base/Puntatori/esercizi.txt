Esercizio 9: 

Riferimento a Puntatore Qual è l'output di questo programma? (Ragiona sul passaggio per valore del puntatore).


void reset(int* p) {
    p = nullptr;
}

int main() {
    int x = 5;
    int* ptr = &x;
    reset(ptr);
    if (ptr == nullptr) std::cout << "Nullo";
    else std::cout << "Non nullo";
}
