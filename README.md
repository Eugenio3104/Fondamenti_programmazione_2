# Fondamenti_programmazione_2
vari esercizi e teoria dell'esame di fondamenti di programmazione 2 unical , informatica 

Cosa prevede il corso : 

1) Introduzione al C++
- Programmazione in C++ - Primi Passi (Struttura di un programma; Concetto di variabile; Espressioni; Precedenza degli operatori; Tipi e conversioni)
- Programmazione in C++ - Strutture di controllo (Istruzioni semplici e composte, definizione di blocco di istruzioni, visibilità delle variabili; costrutti di selezione; costrutti di iterazione)
- Programmazione in C++ - Funzioni (Dichiarazione; Parametri formali e valore di ritorno; Passaggio per valore e passaggio per riferimento)

2) Algoritmi e strutture dati elementari
- Array e Matrici
- Ricorsione e Tecnica Divide et Impera
- Problemi di ricerca e ordinamenti risolti con la Tecnica Divide et Impera
- Puntatori e memoria dinamica

3) Le Classi e la Programmazione Orientata agli Oggetti
- Concetto di classe
- Composizione di classi
- Overloading degli operatori
- Astrazione dei dati e strutture dati complesse
- Classi Template
- Ereditarietà
- Polimorfismo

4) Algoritmi e Strutture Dati Avanzate
- Liste, Pile, Code
- Standard Template Library
- Tabelle Hash
- Alberi
- Grafi
- Tecnica di Programmazione Golosa
- Tecnica di programmazione Dinamica
- Tecnica del Backtracking
